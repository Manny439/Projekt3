PridatZvire: Vytvoří nový objekt zvířete se zadanými údaji a uloží ho do seznamu v paměti.

VypisVse: Projde celý seznam a vypíše informace o všech zvířatech, nebo oznámí, že je útulek prázdný.

VyhledatPodleDruhu: Vyfiltruje a zobrazí pouze ta zvířata, která patří ke konkrétnímu druhu zadanému uživatelem.

VyhledatPodleJmena: Najde a vypíše všechna zvířata, jejichž jméno obsahuje hledaný text.

VyhledatPodleVeku: Zobrazí seznam zvířat, která jsou buď mladší, nebo starší než zadaný počet let.

OznacitAdopci: Vyhledá konkrétní zvíře podle jeho ID a změní jeho stav adopce na „Adoptován“.

Statistiky: Projde všechna zvířata a vypočítá jejich celkový počet, průměrný věk a počet těch adoptovaných.

NacistZakladniData: Slouží k automatickému naplnění seznamu několika testovacími zvířaty hned po startu programu.
